# Changelog

All notable changes to this mod will be documented in this file.

## [1.0.0] - 2026-04-01

### Initial Release
- Allows building on shallow water, chest-deep water, marsh, marshy soil, soft sand, and mud
- Adds Light, Medium, Heavy, and Diggable affordances to restricted terrain types
- Pure XML patches for maximum mod compatibility
- Continued from John Midnight's Build Everywhere (last updated 2018 for RimWorld 1.0)
