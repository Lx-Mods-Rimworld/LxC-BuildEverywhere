# Build Everywhere (Continued by Lexxers)

Allows building on terrain that normally doesn't support it: shallow water, chest-deep water, marsh, marshy soil, soft sand, and mud.

## What it does

Adds Light, Medium, Heavy, and Diggable affordances to terrain types that normally block construction:

- **Shallow water** (fresh, ocean, and moving)
- **Chest-deep water** (moving)
- **Marsh**
- **Marshy soil** (adds Medium + Heavy)
- **Soft sand** (adds Medium + Heavy)
- **Mud**

## How it works

Pure XML patches -- no C# code, no Harmony. Maximum compatibility with other mods.

## Compatibility

- RimWorld 1.6
- Safe to add mid-save
- Compatible with all DLCs
- Uses XPath patches for broad mod compatibility
